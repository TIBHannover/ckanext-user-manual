You can edit a data resource (file) in two ways:

  • On the dataset page, click on the "Explore" button in front of the data resource title. Then, click "Edit"

  • OR, in the dataset page, open the data resource page by clicking on the title. Then, click on "Manage"

Note: You can edit the machine link for all data resources in the dataset edit page. (dataset page --> Manage button --> under "Machines" tab)
