You can access CKAN’s “Create dataset” page in two ways:

  • Select the “Datasets” link at the top of any page. From this, above the search box, select the “Add Dataset” button.

  • Alternatively, select the “organizations” link at the top of a page. Now select the page for the organization that should own your new dataset. Provided that you are a member of this organization, you can now select the “Add Dataset” button above the search box.


CKAN will ask for the following information (metadata) about your data:

  • Title – this title will be unique across CKAN, so make it brief but specific. E.g. “UK population density by region” is better than “Population figures”.

  • Description – You can add a longer description of the dataset here, including information such as where the data is from and any information that people will need to know when using the data.

  • Tags – here you may add tags that will help people find the data and link it with other related data. Examples could be “population”, “crime”, “East Anglia”. Hit the key between tags. If you enter a tag wrongly, you can use its delete button to remove it before saving the dataset.

  • License – it is important to include license information so that people know how they can use the data. If you need to use a license not on the list, contact your site administrator.

  • Organization – If you are a member of any organizations, this drop-down will enable you to choose which one should own the dataset. Ensure the default chosen is the correct one before you proceed.

  • Visibility – Public (anybody can see the dataset) OR
    Private (only the organization members)

  • Source the dataset source if exists (Ex: http://www.example.com/dataset.json)

  • Version the dataset version

  • Author Name and Email

  • Maintainer Name and Email

  • Custom field(s) – Additional information about the dataset if exist

Note: The only required field is the dataset title. However, it is a good practice to provide description, license, and organization.

Note: By clicking on The "Next: Add Data" button, you save the dataset. You can add the data on the next step. For more information you can have a look on the next section.
