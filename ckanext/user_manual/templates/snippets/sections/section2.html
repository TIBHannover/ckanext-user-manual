After saving the dataset information, you can upload or link data to the dataset:

  • You can drag&drop the data files.

  • Alternatively, You can upload them by clicking the "upload" button.

  • Alternatively, You can add the link to the data source by clicking the "Link" button.

Note: If you upload multiple files, the description would be indentical for all of them. You can edit the description later. Please have a look at "How to edit a data resource" section.
